## 内容

AI 正在重塑创业公司的构建方式。本工具覆盖原书四个核心阶段——**想法、MVP、发布、规模化**。

## 交互功能

- 侧边栏常驻导航，滚动自动高亮当前章节
- 章节折叠/展开，阅读进度追踪（localStorage 持久化）
- 退出标准可勾选清单
- 全文搜索（Ctrl+K），支持上下跳转
- 深色/浅色主题切换 + 字号调节
- 文字选中高亮批注，刷新保留
